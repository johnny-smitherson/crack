# Blender Generated Manager Skill

This skill helps agents manage and extend the 3D model and texture asset generation pipeline in the GTA Vice City: Pantelimon codebase.

---

## 🏗️ Folder Structure

- **Manifest Configuration:** `_data/blender_generated/manifest.json`
- **Gateway Manager:** `scripts/generate_assets.py`
- **Model Scripts:** `scripts/models/<name>.py`
- **Generated Assets:** `_data/blender_generated/<name>/`
  - Contains `.blend` project files, `.glb` models, and custom textures.

---

## 🚀 How to Add a New Model

### Step 1: Register in `manifest.json`
Add a new JSON entry to `_data/blender_generated/manifest.json`:
```json
{
  "name": "my_model",
  "input_image": "images/my_image.jpg",
  "script": "scripts/models/my_model.py",
  "blend_path": "_data/blender_generated/my_model/my_model.blend",
  "glb_path": "_data/blender_generated/my_model/my_model.glb",
  "version": "1.0.0",
  "description": "Short description of my model"
}
```

### Step 2: Implement the Model Script
Create `scripts/models/my_model.py`. The script should follow this standard structure:
1. **Pillow Texture Generation (Optional):** If the model needs custom textures drawn based on the photo, implement a helper function using Pillow.
2. **Subprocess Isolation:** To avoid package collision inside Blender's Python (which has PEP 668 restrictions), delegate Pillow operations to the virtual environment python:
   ```python
   subprocess.run(["/tmp/venv/bin/python", "-c", "import PIL..."], check=True)
   ```
3. **Geometry and UV Modeling:**
   - Clear the scene with `bpy.ops.object.delete()`.
   - Build mesh primitives (cubes, cylinders, grids).
   - Create and assign materials with slots.
   - Set UV coords mathematically or use smart project.
4. **Shading and Packaging:**
   - Programmatically set the viewport to Material Preview so it opens correctly:
     ```python
     for area in bpy.context.screen.areas:
         if area.type == 'VIEW_3D':
             for space in area.spaces:
                 if space.type == 'VIEW_3D':
                     space.shading.type = 'MATERIAL'
     ```
   - Pack images using `bpy.ops.file.pack_all()`.
5. **Export:** Export `.glb` and save `.blend` to the target directory.

### Step 3: Run the Generator Gateway
Build the models from the root repository directory:
```bash
# Run all model scripts
python scripts/generate_assets.py

# Run only your specific model
python scripts/generate_assets.py --model my_model
```

---

## 🛠️ Best Practices
- **Self-Contained Files:** Always use `bpy.ops.file.pack_all()` before saving `.blend` files so that textures are embedded and can be viewed instantly by developers.
- **Clean UV Layouts:** Apply textures using normalized mathematical coordinates `[0, 1]` based on vertex coordinates to prevent texture stretching or incorrect rotation.
- **Backup Cleanup:** Delete any `.blend1` backup files generated by Blender inside the output directories before committing to git.
